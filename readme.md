# laravel-version-5.4-Basic-CRUD-
<h3> Download or Clone Repository</h3>
create db with the name of crud

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=crud
DB_USERNAME=root
DB_PASSWORD=
```

# Run migrate artisan command
```
php artisan migrate
```

# Run project
```
php artisan serve
```
